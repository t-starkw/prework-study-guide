HTML

  • HTML is a markup language used to give a webpage structure.
  • HTML elements work together to provide the content and basic framework for all pages on the web.
  • HTML elements can include:

    1. Text elements
    2. Image elements
    3. Header elements
    4. List elements
    5. Meta-related elements
    6. Content division elements
    7. Semantic elements

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS (Cascading Style Sheet) defines how webpage content is presented to users.
  • CSS is used to define:

    1. Colors
    2. Aesthetics
    3. Visual Layout
    4. Organization

  • There are 3 ways to style a webpage with CSS:

    1. Inline CSS: The method of styling an element directly in the HTML file. Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular webpage element. This method is useful for quick and permanant changes.
    2. Internal CSS: The method of embedding an entire style sheet directly into the HTML file using a style element within the head element. These CSS style sheet rules only apply to that page, and a single change to a CSS rule will apply to all of the tagged elements.
    3. External CSS: The method of allowing developers to keep all CSS rules in a separate file, linking that file to the HTML file using a link element placed in the HTML's head element. This is the most common method of CSS styling.

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that allows you to manage and keep track of each version of your project.
  • The command line is used to access Git.
  • Git commands can be used to initiate repositories, add code to a repository, or to view different versions of our code.
  • Some examples of git commands are:

    1. git status: checks what branch we are currently on
    2. git checkout -b: creates a new branch and switches to it
    3. git add -A: adds code to current working branch.
    4. git commit -m: commits files to the current working branch.
    5. git pull origin main: checks to see if the local branch is in sync with the base branch in github.
    6. git push origin: pushes the local changes to the remote github branch.

JavaScript

  • JavaScript is a high-level dynamic programming language that adds interactivity to a webpage.
  • A variable is a named container that allows us to store data in our code.
  • Variables may hold values with a variety of different data types:

    1. String: A sequence of text known as a string enclosed by single quote marks.
    2. Number: This is a number. Numbers don't have quotes around them.
    3. Boolean: A true/false value. The words true and false are special keywords that don't need quote marks.
    4. Array: A structure that allows you to store multiple values in a single reference.
    5. Object: Everything in JavaScript is an object and can be stored in a variable.

  • Conditionals are code structures used to test if an expression returns true or not.
  • Functions are a method of packaging functionality that you wish to reuse. After defining a body of code as a function, that code executes when the function is called.
  • Control flow is the order in which a computer executes code in a script.
